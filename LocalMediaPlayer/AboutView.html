概述

LocalMediaPlayer是一个使用.NET 8.0和C#编写的适用于Win10+的WPF轻量级多媒体应用程序, 具有播放媒体等功能，并且是一个开源免费的应用，可以从Github下载

开始使用

点击连接，按照说明下载最新安装包，或者从源构建。 以下是其支持的操作系统要求

OS最低版本 框架 架构
Windows 7.0 .NET 8.0(长期支持) any CPU

安装完成后，即可使用。

功能介绍

概览

播放本地媒体
播放列表
使用控件交互来控制播放
文件拖拽
使用按键和鼠标键控制
分享功能

点击左上角的组合框，再点击第一个项【打开】，会出现一个打开文件对话框，可以选择多媒体文件。

如果你找不到文件，可以更改右下角的筛选器，
以下是支持的文件格式
音频文件 ".mp3", ".wav", ".ogg", ".flac", ".m4a", ".wma"
视频文件 ".mp4", ".avi", ".mov", ".flv", ".mkv", ".wmv", ".mpg", ".mpeg"

每当使用【打开】成功播放一个文件时，会自动向文件列表添加一个新项（除非已经有这个项了），点击左上角的组合框，然后点击播放列表，即会导航到一个页面， 每个项都包含文件地址，第一次打开的日期，以及一个检查框，如果检查框被选中，那么可以点击右上角的删除按钮进行删除，双击菜单项可以打开文件，点击左上角的返回可以进行返回。


在主窗口下面，有三个按钮，两个滑块，一个菜单，可以进行快进，倒退，暂停，播放，调节进度，调节音量，菜单项可以调节播放速度，或者分享。 以下是支持的倍速

0.25x
0.5x
0.75x
1.0x
1.5x
2.0x
注意，当拖动进度滑块时，媒体会暂停，画面也会固定，停止拖放时点击按钮继续播放。


你可以把符合要求的文件拖拽到主窗口，这样我们就会为你播放该文件


在主窗口中，如果应用具有焦点，可以使用按键和鼠标键控制播放，以下是键表

键值 功能
mouse-double-click 暂停或播放媒体
left 回退30秒（如果有）
right 快进30秒（如果有）
space 暂停或者播放
鼠标必须在视频窗口进行双击


在主窗口左下角的【操作】菜单，点击【分享】，会有三种分享方式，【本地分享】，【网络分享】，【邮件分享】。以下是详细内容。

功能名称 简述 备注
本地分享 将正在播放的视频复制到指定目录
网络分享 使用套接字即时分享当前视频 应用程序在分享时对方必须已经处于等待状态，对方必须下载【接收分享】程序，点击此下载， 注意，在点击网络分享时，原来的视频会被关闭
邮件分享 根据你的邮箱（需要你的密码，我们尊重你的隐私）以及对方的邮箱，通过发送带有标题，正文和附件（当前视频）的邮件分享
